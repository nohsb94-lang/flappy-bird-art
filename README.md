# Flappy Bird Art

이 프로젝트는 Helpy Chat(헬피 챗)과 GitHub MCP를 활용하여 만든 커스텀 Flappy Bird 웹 게임입니다.

- **게임 특징**
  - 아름답고 미려한 새 디자인 (빨간 부리, 하얀 몸통, 파란 날개, 세 갈래 꼬리)
  - 장애물: 위는 고드름(얼음), 아래는 나무 기둥
  - HTML5 Canvas와 JavaScript로 제작

## 시작하기

1. 이 저장소를 클론하거나, [GitHub Pages](#github-pages-배포-방법)를 통해 바로 플레이할 수 있습니다.
2. 직접 실행하려면:
    ```bash
    git clone https://github.com/nohsb94-lang/flappy-bird-art.git
    cd flappy-bird-art
    python3 -m http.server 8000
    ```
    브라우저에서 http://localhost:8000 접속

## GitHub Pages 배포 방법

1. 저장소의 `Settings` > `Pages` 메뉴로 이동합니다.
2. "Branch"를 `main`으로, 폴더는 `/ (root)`로 선택 후 저장합니다.
3. 잠시 후 `https://{사용자명}.github.io/flappy-bird-art/` 주소에서 게임을 플레이할 수 있습니다.
